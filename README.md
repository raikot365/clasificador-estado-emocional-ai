# Clasificador del Estado Emocional Basado en Entradas Textuales
El procesamiento del lenguaje natural (NLP) aplicado a la salud mental es un campo en rápido crecimiento con el potencial de mejorar significativamente la accesibilidad y eficacia de los servicios de apoyo psicológico. El dataset "_Mental Health Dataset_" proporciona una colección estructurada de entradas de usuario y respuestas relacionadas con diversas condiciones de salud mental, diseñado para facilitar el desarrollo de chatbots y aplicaciones de asistencia virtual en este ámbito.
El objetivo principal de este trabajo es desarrollar un sistema de procesamiento de lenguaje natural capaz de:
 1. Clasificar el estado emocional o la condición de salud mental del usuario basándose en sus entradas textuales.
 2. Determinar la necesidad de mejorar o no la calidad/cantidad de ejemplos en el dataset, para desempeñar la tarea de manera óptima.
